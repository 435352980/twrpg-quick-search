# twrpg-quick-search

世界 RPG 装备速查

## 调试

### 1.请先确保系统拥有 nodejs 运行环境

### 2.安装依赖

```
cd <项目根目录> 命令行运行 yarn install
```

### 3.启动页面

```
yarn dev
```

### 4.启动 electron 调试

```
yarn start
```
